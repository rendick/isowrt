# isowrt

Easy way to burn a bootable flash drive with ISO.

# Available OS

**UNIX systems with preinstalled GNU coreutils are supported.**

# Install

**Recommended**

```bash
git clone https://github.com/rendick/isowrt.git
cd isowrt
chmod +x isowrt.sh
./isowrt.sh
```
