# isowrt

Easy way to burn a bootable flash drive with ISO.

# Available OS

**All UNIX systems are supported.**

# Install

```bash
git clone https://github.com/rendick/isowrt.git
cd isowrt
chmod +x isowrt.sh
./isowrt.sh
```

**Soon I will make a PKGBUILD file for Arch-based distributions.**
